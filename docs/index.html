One command. Any app. Full AI control.

appctl introspects any web app, database, or service and turns it into a sandboxed tool layer for any LLM. 

cargo install appctl

Sources

  • --openapi OpenAPI / Swagger
  • --rails Ruby on Rails
  • --laravel Laravel
  • --aspnet ASP.NET Core
  • --strapi Strapi
  • --supabase Supabase / PostgREST
  • --django Django
  • --db PostgreSQL / MySQL
  • --url Live URL + login / CSRF
  • --mcp MCP servers
  • --plugin Dynamic plugins

Use it

appctl sync --openapi https://api.example.com/openapi.json
appctl chat
appctl run "Add a user named John"
appctl serve --port 7878

Providers

Any OpenAI-compatible endpoint plus Anthropic: OpenAI, OpenRouter, NVIDIA NIM, Groq, Together, Fireworks, Ollama, LM Studio, vLLM, LiteLLM, Anthropic. Secrets live in the OS keychain.

Plugins

Build a cdylib against appctl-plugin-sdk and install it: 

appctl plugin install <path|crate|git-url>
appctl plugin list

Safety

  • --read-only blocks writes.
  • --dry-run previews without executing.
  • Mutations prompt by default.
  • Every tool call is logged to a local SQLite audit DB.
